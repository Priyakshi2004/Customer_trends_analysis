Customer Behavior Analytics Dashboard
Overview
This project analyzes customer shopping behavior using Python, SQL, and Power BI. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard development, and business reporting. The objective is to uncover customer purchasing patterns, subscription behavior, spending trends, and product preferences.

Dataset

The dataset contains customer shopping and purchasing information, including:
Customer ID
Age Group
Gender
Category
Purchase Amount
Subscription Status
Review Rating
Purchase Frequency
Previous Purchases
Payment Method
Location
Season
Shipping Type
Dataset File: customer_shopping_behavior.csv

Project Workflow
1. Data Loading
Imported dataset into Python using Pandas.
Verified data structure and column types.

2. Exploratory Data Analysis (EDA)
Analyzed customer demographics.
Examined purchasing behavior.
Studied subscription trends.
Identified spending patterns.

3. Data Cleaning
Checked for missing values.
Removed inconsistencies.
Standardized categorical fields.
Converted data into analysis-ready format.

4. SQL Analysis
Performed SQL queries to answer business questions such as:
Customer distribution by category
Revenue by product category
Subscription status analysis
Average customer ratings
Purchase frequency trends

5. Dashboard Development
Created an interactive Power BI dashboard featuring:
Total Customers
Average Purchase Amount
Average Review Rating
Subscription Status Distribution
Revenue by Category
Interactive Filters and Slicers

Key Findings 
Majority of customers are non-subscribers.
Certain product categories generate significantly higher revenue.
Purchase frequency impacts overall customer value.
